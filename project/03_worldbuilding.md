# Worldbuilding

## Setting Summary
現代日本によく似た世界。一般市民の日常と、サトーを守る国家規模の秘匿警護任務が同時進行している。今回は春の日曜日、東京側の生活圏で、馴染みのパン屋から夜のベランダまでが一本の休日導線になる。

## Key Background Concepts
- 今回の背景 01: 早朝のマンション室内と玄関先。柔らかい朝日、支度中の服、まだ人の少ない共用廊下
- 今回の背景 02: 馴染みのパン屋と併設喫茶店。焼きたての棚、黒い珈琲、納品口のある朝の店先
- 今回の背景 03: 川沿いの古本街。影のあるアーケード、古い木枠、紙の匂いがありそうな棚影
- 今回の背景 04: 近所の商店街から白鷺川沿いベンチへ抜ける昼の導線。団子屋、のれん、河川敷の手すりとベンチ
- 今回の背景 05: 近所のスーパーの夕方。惣菜棚の明かり、買い物かご、配送口、住宅街へ抜ける出口
- 今回の背景 06: 夜更けのリビングから続くベランダ。スタンドライト、レコード、食後の皿、都心の小さな灯り

## Rules
- 魔法・技術: 超常要素はない。すべて現代的な諜報、拉致工作、事故偽装、通信妨害、拘束具で成立する
- 社会: 平和な日常の奥で、国家安全保障の最深部だけがサトーの扱いに神経を尖らせている
- 国家機密: 防衛の切り札となる極秘システムは、最終起動に登録済み本人の全身生体認証を必要とする
- 隠蔽構造: 登録事故は公表されておらず、政府内部は非常時だけ秘密裏にサトーを確保すればよいという危うい運用で問題を棚上げしている
- 国際情勢: 第三国はこの秘密を把握しており、サトーを生きたまま拉致して使用不能にすることを狙う
- 叙述方針: 国家機密は設定上の正本として保持し、本文では直接説明しすぎない
- 地理: 東京側生活圏。白鷺川、馴染みのパン屋と併設喫茶店、川沿い古本街、近所の商店街、スーパー、帰宅圏のマンション
- 季節: 春の晴れた日曜日。風と光で気分の良さが立ち、夜はまだ少し冷える
- 建築: 個人経営のパン屋と喫茶店、古い個人店アーケード、生活感のある商店街、中層マンションのベランダ生活
- 禁則: 銃撃戦、爆発、群衆パニックのような大騒ぎは起こさない。すべては「休日の裏で静かに処理された」に収める

## Sensory Notes
- 色: 朝は薄金、昼は白と青、夕方は琥珀、夜は紺と暖色の室内灯
- 質感: シャンブレー、デニム、紙袋、木の手触り、川風、ベンチの金属、レコードジャケットの紙
- 音: 共用廊下の金属音、パン屋の扉、喫茶店のカップ、ページをめくる音、商店街の遠い呼び込み、レコードの針
- 匂い: ブラックコーヒー、焼きたてのパン、古本の紙、みたらし団子の甘い焦げ、惣菜の湯気、夜の室内の静けさ
